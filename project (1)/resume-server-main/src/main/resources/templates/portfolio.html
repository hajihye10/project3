IoT & Database

스마트 환경 모니터링 시스템 구축

Arduino를 활용한 실시간 센서 데이터 수집 및 SQL 연동

**개요:** 온도/습도 센서 데이터를 Arduino로 수집하고, 이를 PC로 전송하여 데이터베이스에 저장 및 웹으로 모니터링하는 시스템을 구축했습니다.

핵심 기여 및 기술

  • **하드웨어 인터페이스 설계:** Arduino와 센서 간의 통신 프로토콜 구현.
  • **데이터베이스 구조화:** 센서 데이터를 효율적으로 저장 및 관리하기 위한 SQL Lite 스키마 설계.
  • **SQL 활용:** 기초적인 `INSERT`, `SELECT` 구문을 활용하여 데이터 CRUD 기능 구현.
Network & System

소규모 오피스 네트워크 토폴로지 설계 및 구현

라우터 및 스위치 장비를 이용한 실습 중심의 네트워크 구성

**개요:** 3개의 부서(총 50명 규모)를 위한 LAN 환경을 가상으로 설계하고, 네트워크 장비 기본 설정을 통해 통신 환경을 구축했습니다.

핵심 기여 및 기술

  • **IP 주소 계획:** 효율적인 자원 분배를 위한 서브넷팅 실시 및 IP 대역 할당.
  • **라우터/스위치 설정:** VLAN, Port Security, Static Routing 등 핵심 **네트워크 장비 기본 설정** 명령어 적용.
  • **OSI 7계층 이해:** 각 계층별 프로토콜의 동작 원리를 실제 구성에 반영.
Circuit & OS Foundation

디지털 회로 및 컴퓨터 구조 기초 실습

하드웨어와 시스템 소프트웨어 기초 이해를 위한 실습 통합 프로젝트

회로 기초 실습

  • 기본 논리 게이트(AND, OR, NOT)를 활용하여 간단한 디지털 회로(가산기 등) 구성 및 작동 검증.
  • 반도체 소자의 기본 특성과 작동 원리에 대한 이해 심화.

컴퓨터 구조 및 운영체제 기초

  • 컴퓨터의 5대 구성 요소와 CPU 동작 원리 이해.
  • 운영체제의 역할, 특히 프로세스 관리 및 메모리 관리 개념 학습.
  • 간단한 스케줄링 알고리즘(FIFO, SJF)의 개념을 이해하고 시뮬레이션으로 구현 시도.

본 포트폴리오는 하지혜 지원자의 정보통신공학 역량을 보여주기 위해 제작되었습니다.
